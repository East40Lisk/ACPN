# ACPN Core

This repository defines the **canonical specifications** for the Autonomous Content Protection Network.

If it is not defined here, it is not ACPN.
# Automated Creator Protection Network (ACPN)

ACPN is an open‑source framework to automate protection of digital creators’ rights,
with cross‑platform enforcement, privacy‑preserving proofs, and on‑chain verifiability.

## Modules

| Folder | Purpose |
|--------|---------|
| `acpn-core/` | Schemas, RFCs, test vectors |
| `acpn-wasm-enforcer/` | Rust → WASM enforcement engine |
| `acpn-zk-quorum/` | Zero‑knowledge circuits and tooling |
| `acpn-js-sdk/` | SDK for JS/Node/Browser/Termux |
| `acpn-contracts/` | Solidity contracts for on‑chain enforcement |
